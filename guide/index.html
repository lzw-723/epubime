使用指南 ​

欢迎来到 EPUBime 的使用指南！EPUBime 是一个纯 Java 库，用于解析 EPUB 文件格式，支持 EPUB 2 和 EPUB 3 格式。

简介 ​

EPUBime 提供了完整的 EPUB 文件解析功能，包括元数据提取、章节内容读取和资源文件处理。该库具有以下特性：

  • 完整的 EPUB 支持：支持 EPUB 2 和 EPUB 3（包括 EPUB 3.3 规范）格式解析
  • 高性能：智能缓存机制和流式处理，优化内存使用
  • 完善的异常处理：详细的异常分类和错误信息，便于调试和错误定位
  • 丰富的功能：元数据提取、章节解析、资源文件处理、目录解析等
  • 简洁的 API：易于使用和集成的 API 设计
  • 高质量代码：严格的测试覆盖率和静态代码分析
